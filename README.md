# Daily Tasks

A simple iOS app for managing daily tasks, built with SwiftUI.

## Requirements

- Xcode 16+
- iOS 18+
- Swift 6

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/daily-tasks.git
   ```
2. Open `daily-tasks.xcodeproj` in Xcode.
3. Build and run (⌘R).
